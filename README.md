## Instalação
```bash
# Clone do Repositorio
$ git clone https://github.com/WesleyReboucas/SmartBank.git
```

### Web

```bash
# Na pasta SmartBank
$ cd SmartBank

# Instalação de depedencias
$ yarn install

# Iniciando aplicação
$ yarn start
```
**Acesse:** http://localhost:3000/
